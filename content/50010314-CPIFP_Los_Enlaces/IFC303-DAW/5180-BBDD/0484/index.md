# Sistemas Informáticos (0484)

Bienvenido a la documentación del módulo **Sistemas Informáticos** del ciclo **DAW**.

## Contenidos

- Hardware de sistemas informáticos
- Software de sistemas informáticos
- Instalación de sistemas operativos
- Gestión de información y recursos en redes
- Configuración de sistemas operativos
- Explotación de aplicaciones informáticas

## Objetivos

1. Identificar la estructura y los componentes funcionales de un computador.
2. Instalar, configurar y explotar sistemas operativos.
3. Gestionar la información y los recursos de una red.
