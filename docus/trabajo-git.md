# Fórmula de trabajo con Git asociado a materiales

Los materiales que componen el directorio `content` provienen de un repositorio git. 

 a. Repositorio Ministerio
 b. Repositorio Aragón
    b.1. Rama oficial 
    b.2. Ramas de usuarios

* Cuando un profesor accede por primera vez, se le crean ramas propias de todos los módulos que imparte [`DARIO` quizá pueda elegir de qué materiales quiere tener rama dentro de su especialidad]

* Todo profesor puede ver:
  * Rama del Ministerio
  * Rama oficial Aragón
  * Ramas propias

* Como profesor puedo modificar sus `ramas propias` y guardar esos cambios.
* Como profesor puedo descargar como pdf mis desarrollos
